# AI Studio

Локальная AI-среда разработки с использованием Ollama.

AI Studio — это open-source проект, цель которого — создать модульную систему искусственного интеллекта для разработки программного обеспечения.

В отличие от большинства AI-приложений, AI Studio не рассматривает языковую модель как всё приложение целиком. Модель — лишь одна из частей системы.

Основная идея проекта — построить полноценную локальную среду разработки, которая понимает структуру проекта, анализирует код, хранит контекст, использует инструменты разработки и постепенно автоматизирует весь процесс создания программ.

Вся работа выполняется локально на вашем компьютере.

> **Статус проекта:** Активная разработка • Core v0.1 Alpha

---

# Почему AI Studio?

Большинство локальных AI-приложений представляют собой обычный чат.

AI Studio строится по другому принципу.

Вместо одного большого промпта здесь используется набор независимых компонентов, каждый из которых отвечает только за свою задачу.

Например:

- анализ структуры проекта;
- хранение истории диалога;
- понимание архитектуры проекта;
- планирование работы;
- генерация кода;
- использование инструментов разработки.

Такой подход позволяет постепенно превратить AI Studio из обычного чата в полноценную инженерную среду.

# Архитектура

```text
                          Пользователь
                                │
                                ▼
                           Conversation
                                │
                                ▼
                              Session
                                │
                                ▼
                                LLM
                                │
                                ▼
                              Ollama


                              Project
                      ┌─────────┼─────────┐
                      ▼         ▼         ▼
                 Analyzer   Scanner    Brain
```

Архитектура проекта построена модульно.

Каждый компонент отвечает только за одну задачу и может развиваться независимо от остальных.

AI Studio создаётся как полноценная локальная среда разработки, которая сможет:

- понимать существующие проекты;
- хранить знания о проекте;
- планировать разработку;
- писать код;
- проверять собственные изменения;
- работать с Git;
- использовать инструменты разработки;
- работать полностью локально.
